# Next Architecture

After the MVP loop works, evolve the system in this order.

## Version 1: Operational Loop

- n8n schedule
- API-based job collection
- OpenAI scoring
- Google Sheets ledger
- Zoho CRM deal creation

## Version 2: Better Data

- Add Greenhouse and Lever source branches.
- Add company enrichment.
- Add recruiter/contact extraction.
- Add duplicate detection against Zoho before creating deals.
- Add daily report generation.

## Version 3: Application Engine

- Tailored resume angle.
- Cover note / recruiter message draft.
- LaTeX resume variant generation.
- Follow-up task automation.
- Outcome tracking.

## Version 4: MCP Action Layer

- Query Zoho and Sheets in natural language.
- Create or update Zoho tasks from natural language.
- Generate application packets from selected opportunities.
- Search semantically across historical companies, applications, and postings.

## Version 5: Learning System

- Store application outcomes.
- Compare score predictions against real interview results.
- Tune scoring weights.
- Build company/recruiter memory.
- Rank opportunities by expected value, not just surface fit.
